# Machine Learning by Python
常见机器学习算法的Python实现。  
详细文档请[点击这里](https://wisedoge.github.io/ML-by-Python/mlearn.html)

## 内容
* K近邻算法(KNN)
* 感知机
* 朴素贝叶斯分类器
* 决策树(Decision Tree)
* 随机森林(Random Forests)
* 支持向量机(Support Vector Machine)
* 线性回归(Linear Regression)
* 逻辑斯蒂回归(Logistic Regression)
* Bagging算法
* 神经网络(ANN)
* 隐马尔科夫模型(HMM)
* K-Means聚类算法
* LVQ聚类算法
* PCA主成份分析法

mlearn.bagging module¶

class mlearn.bagging.Bagging(basics)¶

Bases: object

Bagging算法. Bagging是一种集成学习算法，学习的本身依赖于基学习算法。 本算法使用时需先实例化N个基学习器：

clfs = [KNeighborClassifier(2) for i in range(N)]
并且基学习算法必须重载了
